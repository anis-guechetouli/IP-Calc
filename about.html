IP Calc

L'outil ultime pour les étudiants en réseau et les administrateurs système.

Pourquoi ce projet ?

Ce projet a été conçu dans le cadre d'études en informatique (BTS SIO / Réseaux). Le subnetting (calcul de sous-réseaux) est une compétence fondamentale mais souvent fastidieuse à faire à la main. IP Calc vise à fournir une interface moderne, rapide et précise pour vérifier ses calculs et comprendre la structure binaire des adresses IP.

Fonctionnalités

  • Calcul IPv4 complet (CIDR)
  • Visualisation binaire
  • Wildcard Mask (Cisco ACL)
  • Détection OSINT (GeoIP, ISP)
  • Mode Sombre persistant

Tech Stack

HTML5 Tailwind CSS (CDN) Vanilla JS API Fetch

Aucun framework lourd (React/Vue) n'a été utilisé. Le site est ultra-léger et fonctionne directement dans le navigateur.

FAQ

Est-ce que mes données sont enregistrées ?

Non. Tout le calcul se fait en local dans votre navigateur via JavaScript. Seule la fonctionnalité OSINT envoie l'IP à une API publique (ipwho.is) pour récupérer la géolocalisation.

Qu'est-ce que le Wildcard Mask ?

C'est l'inverse du masque de sous-réseau. Il est principalement utilisé dans les configurations de routeurs Cisco (OSPF, ACL) pour définir des plages d'adresses.

Pourquoi l'IPv6 est-il limité ?

L'IPv6 est beaucoup plus complexe (128 bits). Ce calculateur se concentre sur l'IPv4 pour l'apprentissage du subnetting classique, mais fournit les infos de base pour l'IPv6.
